# Gintaro Kapočiaus baigiamasis darbas

## Reikiamos programos

- Node.js
- Naršyklė

## Instaliavimas

1. Direktorijoje `client`, įvykdykite komandą:

- npm install

2. Direktorijoje ` server`, įvykdykite komandą:

- npm install

## Projekto paleidimas

1. Direktorijoje `client`, įvykdykite komandą:
   - npm start
2. Direktorijoje `server`, įvykdykite komandą:
   - npm start
